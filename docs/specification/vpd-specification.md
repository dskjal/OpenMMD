# vpd ファイルのファイルフォーマット

- 拡張子は vpd
- エンコーディングは Shift-JIS
- 改行は CRLF
- `//` とその後の文字はコメントとして無視される
- `;` とその後ろの文字もすべて無視して問題ない
- 座標は左手系
- OpenMMD ではローダー側で右手系へ変換する

1行目: `Vocaloid Pose Data file`
2行目: 空改行
3行目: モデル名+.osm+;。例 `初音ミク.osm;`
4行目: 総ボーン数+;。例 `100;`
5行目: 空改行

以下ボーンリスト。

## ボーンリスト

Boneボーンカウント{ボーン名
  実数X,実数Y,実数Z;    // Pos
  実数X,実数Y,実数Z,実数W;  // Quaternion x, y, z, w
}
空改行
